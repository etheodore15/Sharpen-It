# SharpenIt

Australia's Sharpest Postal Service — [sharpenit.com.au](https://sharpenit.com.au)

Live site: https://etheodore15.github.io/Sharpen-It/

Admin panel: https://etheodore15.github.io/Sharpen-It/#/admin (password: `sharp2026`)

## Stack

- React 18 + Vite
- Tailwind CSS v3
- React Router DOM v6 (HashRouter for GitHub Pages compatibility)
- All data stored in localStorage — no backend required

## Dev

```bash
npm install
npm run dev
```

## Deploy

Auto-deploys to GitHub Pages on every push to `main` via `.github/workflows/deploy.yml`.
